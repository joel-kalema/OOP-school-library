## Getting Started

```bash
  git clone git@github.com:joel-kalema/my_Enumeration.git
  cd into the folder by typing:
  - cd person.rb
  - cd student.rb
  - cd teacher.rb
```

open your code editor and run

```bash
  bundle install
```
